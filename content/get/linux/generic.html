Download Lakka

This step is independent of your OS choice, the downloaded image can be used on any OS.

However, the image is specific to the target hardware, as its name should reflect it.

Choose one of the following images depending on your PC architecture.

Download Lakka for 32 bits CPU ~ 400MB

Download Lakka for 64 bits CPU ~ 400MB

Download Lakka for 64 bits CPU, Vulkan on Nvidia GPUS only ~ 400MB

Please note that due to the wide variety of PC hardware, Lakka may not work on your hardware.

Those image are USB images, not CD or DVD images, those installation mediums are unsupported. Also, we don't support virtualization and dualboot. Lakka is meant to be installed on real hardware.

The "Vulkan on Nvidia GPUS only" version has less cores than the regular 64 bits image, but some heavier cores will run better.
